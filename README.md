# 🔐 Password Manager

A secure command-line password manager built with Python from scratch.

## ✨ Features
- 🔐 Master password with SHA-256 hashing
- ➕ Add, View, Search & Delete passwords
- 🎲 Random password generator
- 💪 Password strength checker
- 🔒 AES Fernet encryption — passwords stored as gibberish not plain text

## ▶️ How to Run
pip install cryptography
python password_manager.py

## 🛠️ Built With
- Python
- cryptography
- hashlib
- json
